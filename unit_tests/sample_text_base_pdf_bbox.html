LibeRustOffice What is LibeRustOffice? LibeRustOffice is a Rust-native desktop .odt editor built with egui, aiming to provide a lightweight Writer-style document workflow with import, page-like rendering, basic rich-text editing, export back to .odt, and strong logging/testing/screenshot-regression infrastructure, while improving document fidelity incrementally rather than targeting full LibreOffice feature parity upfront. • Item 1 • Item 2 • Item 3 Enumerate test: 1.
